{#
Configured colors

By defining a color in your configuration, you can access that color by its key value.

These keys also accept modifiers that allow you to access tints, shades, and opacities of the color you defined. For example, if you defined your main brand color as "primary", you can access up to three shades darker by using "dark.primary", "darker.primary", and "darkest.primary".

Opacity works in a similar way using a numeric value. For example, you can access a 50% opacity value by using "primary.5". Similarly, you can also access opacity values for modified colors as well, using both modifiers together: "lighter.primary.2".
