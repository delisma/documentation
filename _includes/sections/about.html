Hydrogen is a design and development tool for building coherent web interfaces without writing any CSS.

Here are the core concepts behind Hydrogen:

  • Hydrogen uses HTML5 custom data-attributes to allow you to style your web project.
  • It is a utility style tool that generates a CSS file based on a JSON configuration file.
